API Project: Timestamp Microservice - Java, Spring Boot

User Stories:

  1. I can pass a string as a parameter, and it will check to see whether that string contains
    either a unix timestamp or a natural date (format 'yyyy-MM-dd')
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns an error message
    and a BAD_REQUEST status code.

Example Usage

  • /api/timestamp/2015-12-25
  • /api/timestamp/1451001600000

Example Output:

{ "unix": 1451001600000, "utc": "Fri Dec 25 00:00:00 GMT 2015" }
